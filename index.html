Introduction:
·A motherboard is the fundamental printed circuit board (PCB) in a computer. All components and external peripherals are connected through the motherboard.
·Random access memory(RAM) is the hardware in a computing device where the operating system (OS), applications, and currently used data are stored so that the device's processor can reach it quickly. RAM is the primary memory in a computer.

Motherboard

Motherboards can be found in almost all computers, especially desktop and notebook PCs. the components connected through them include chipsets, central processing units (CPUs) and memory. Peripherals include Wi-Fi, Ethernet, and graphics cards with graphics processing units, or GPUs

Gaming motherboards are made for high performance computers; they are more powerful and versatile than motherboards for desktop and laptop computers.

RAM

Random Access Memory It is one of the most basic elements of computing.

RAM is a temporary memory bank where your computer stores the data it needs to retrieve quickly. RAM makes data easily accessible, so your processor can find it quickly without having to go into long-term storage to complete immediate processing tasks.

Every computing device has RAM, whether it's a desktop computer, a tablet or smartphone, or a smart TV.
